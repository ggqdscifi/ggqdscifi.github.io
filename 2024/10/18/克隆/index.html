克隆

发表于2024-10-18更新于2024-10-18

克隆

内曼2024-10-182024-10-18

参考资料：
B站Nothing教研所-C4D运动图形
CINEMA 4D 2024用户手册

核心作用

把对象批量复制，形成集群，得到一个造型。

克隆是用来摆造型的，不是用来K动画的，K动画可以交给效果器和域来做。

共同特性

4+4种分布形式

所谓分布形式，就是比如把整个集群当成方队，分布形式就是队形的基本样子，也就是最终造型的底子。

克隆的造型本质上都是按某种对象的结构来分布的，分为4种：多边形、样条、粒子、矩阵。

除此之外，还预设了4种既定的分布形式：网格、蜂窝、线性、放射。（本质上还是按多边形、样条对象的结构分布的，只不过它们比较常用，所以另做预设）

4类调整造型的属性

确定好造型的基础底子后，克隆通过4类属性，来进一步调整集群的造型（调整的是方队整体）。

  • 造型基础：当前分布形式因为本身特征而自带的基本属性。（比如，放射形式肯定就有“放射的半径”、蜂窝形式肯定就有“蜂窝的错落程度”）
  • 集群数量：集群的总数。
  • 造型变换（个体关系） ：集群个体之间基础变换的关系。（队员之间的间距、瞅哪、站哪等等）
  • 造型变换（变体） ：当前形式造型的变体。（在队形上额外搞搞花样）

3类通用的属性

除了调整造型，克隆还有3类通用的属性，用来在个体层面上做些调整（调整的是放队里的个别队员）。

  • 当个体对象不止一种时，集群个体的排序方式。（不同队员之间怎么排序，比如个高的和个矮的交替着站）
  • 总体控制个体的基础变换、纹理投射方式。（让不让队员自己搞花样）
  • 集群的渲染、显示方式，这个会影响渲染速度。（队员反应快不快）

关于Offset的参数

遇到偏移Offset往往会带一堆参数。如果有一堆参数里有个是Offset（往往代表一堆家伙的位置偏移），那大概率Offset会带一个Variation（这堆家伙位置偏移的随机程度），Variation又会带一个Seed（随机程度的对应种子值）。另外Offset可能会带一个动画版参数（比如Rate）。把它们看成一起的就行，它们是用来增加复杂细节的。

Object选项卡的参数详解

版本2024.4.0。

通用参数

  • Mode
    集群的分布形式。对应的是4+4种分布形式。

  • Clones
    （当克隆下的子级对象有不止一种时）集群的排序方式。对应的是个体的排序方式。

    • Iterate
      根据层级顺序分布，一轮一轮的循环排序。

    • Blend
      逐渐过渡。

      • 如果子级对象的顶点数相同并且参数也一样，就会按层级顺序分布（就像Iterate那样），同时呈现慢慢过渡的样子。
      • 如果子级对象的顶点数不同，或者参数不一样，也会按层级顺序分布，（但跟Iterate不一样，而是在单一轮排序中进行填充式的分布），同时各子级对象的数量会尽量平均。

    • Sort
      全部显示为某一子级对象，并且可切换。

      • 默认状态下全部显示为层级中首个子级对象，这时可以用任一效果器Parameter选项卡下的Modify Clone来控制切换。
        比如，子级有正方体、球、胶囊这3个对象，Modify Clone值为0-33%时显示为正方体，34-66%时显示为球，67-100%时显示为胶囊。也就是说，数值的对应取决于均分的子级个数。
      • 其实，即使设为Iterate、Random、Blend，效果器的Modify Clone仍然可以这样控制，区别在于，当值为0%时会按Iterate、Random、Blend进行排列，而当值为1-100%时就会按Sort那样起作用。

  • Reset Coordinates
    接管子级对象的基础变换属性。对应的是个体的基础变换。
    比如，如果子级对象已经调节过自身Coordinates选项卡中的位移、旋转、缩放参数，那么勾选Reset Coordinates会让子级对象在这些变换属性上归零，它们k过的动画也会失效，此时调节子级对象的变换参数也不会有任何效果了，而是需要到克隆的Transform选项卡中才能调节。
    注意，取消勾选Reset Coordinates时，克隆Transform选项卡中已经调过的参数仍会起作用。

    • 如果想在保持勾选Reset Coordinates时仍然让子级对象的变换属性及动画有效，也有办法，就是把子级对象打组后放在克隆下，因为Reset Coordinates只会对克隆下的那一级子级发生作用，打组后组里什么样克隆是不管的。

  • Fix Texture
    克隆的材质在集群上的显示方式。对应的是纹理投射方式。
    前提：当克隆自身有材质，并且材质Tag选项卡下的Projection设成Flat。

    • Off
      材质固定在克隆上，不固定到集群，仅投影到集群。
      比如，对克隆添加效果器使集群发生位移时，材质并不会跟着集群的每个个体一起位移。
    • Straight
      材质固定到集群。
      比如，当集群发生位移时，材质就像固定在集群的每个个体上跟着位移。

  • Instance Mode
    实例模式，用于集群数量太多时加速渲染。对应的是集群的渲染方式。
    优先选Multi-Instance和Render Instance，其次选Instance。

    • Instance
      集群的每个个体都是单独渲染。
    • Render Instance
      渲染集群的一个个体，然后把结果复制粘贴到其他个体。
    • Multi-Instance
      在渲染实例的基础上减小内存消耗，更适用于克隆成千上万的植被。

  • Viewport Mode
    视窗模式，切换克隆集群的显示方式来加速预览。对应的是集群的显示方式。
    前提：Instance Mode设为Multi-Instance。

    • Off
      不显示。
    • Dots
      用圆点代替。
    • Matrix
      用矩阵方块代替。
    • Bounding Box
      显示为边框体积，并且不受到光影、材质的影响。
    • Object
      正常显示。

独有参数

图中红色、黄色、绿色、蓝色框分别对应4类调整造型的属性。

当Mode设为Grid时

对应的是网格分布。

网格分布没有造型基础的参数，因为它最基础。

  • Mode
    集群整体尺寸设定的依据。

    • Per-Step
      按相邻个体的间隔。
    • Endpoint
      按首尾两个个体的距离。

  • Form
    给网格集群整体赋予一个预设的外形。
    其中Object以指定多边形对象为外形，与Volume Effector效果器的结果相同。

当Mode设为Honeycomb时

对应的是蜂窝分布。

当Mode设为Linear时

对应的是线性分布。

  • Offset
    这个Offset不是常规的偏移，是指从第几个个体的位置开始克隆（克隆下有多种子级个体时更容易看出来）。

  • Amount
    沿线性走向的完成度。
    值为0%时，集群所有的个体会重合在线性走向的起点，并且变换属性相同；值逐渐增加倒100%的过程中，所有的个体会在位移、旋转、缩放上逐渐演变为下方PRS变换属性已设定的值。

  • Step Mode
    集群旋转设定的依据。

    • Single Value
      相邻两个个体的旋转值是固定的。
    • Cumulative
      后一个体以前一个体的旋转值为基础做指数增长，直观上卷的越多旋转幅度越大。

  • Step Rotation
    集群旋转的值。

当Mode设为Radial时

对应的是放射分布。

放射分布没有造型变换（变体） 的参数。

  • Align
    集群个体是否统一朝向放射中心。

当Mode设为Object时

展开Object旁边的>可以直接调节被指定对象的自身参数。

目标指定为多边形对象

对应的是多边形分布。

多边形分布自身没有造型变换（个体关系） 和造型变换（变体） 的参数，但可以等效替代为调节目标多边形对象的点线面结构和自身属性。

  • 朝向

    • Align Clone
      个体的朝向是否与多边形结构走向相切。
    • Up Vector
      与多边形结构走向相切时，个体朝向所围绕的轴向。
      前提：勾选Align Clone，并且Distribution设为Vertex、Polygon Center、Surface。（也就是当集群被固定到多边形结构的特定点位，而且紧贴结构走向的时候，才有调整朝向的需要）

  • 分布方式

    • Distribution
      集群分布于多边形的什么结构上。
    • Selection
      让集群在指定范围内分布。

目标指定为样条对象

对应的是样条分布。

样条分布自身没有造型变换（变体） 的参数，但可以等效替代为调节目标样条对象的点结构和自身属性。

  • 第二导轨

    • Rail
      指定另一个样条对象，作为改变集群朝向或缩放的参照。
      指定后，集群会以原样条对象为轴，扭向导轨样条对象。（前提：勾选Align Clone）
    • Target
      集群是否摆脱原样条对象轴向的约束，精确朝向导轨样条对象。（勾选的话，集群就必须以样条走向为轴）
      前提：勾选Align Clone。
    • Scale
      当与导轨样条对象的顶点的距离发生变化时，集群是否会因此缩放。
      这个不需要勾选Align Clone。

  • 分布方式

    • Distribution

      • Count
        按样条上顶点的密度，顶点密集的地方集群也越密集。
      • Step
        按相邻个体之间的固定间隔。
      • Even
        按集群总数均匀分布。

    • Per Segment
      当样条对象包含多个样条时，是否对其中每个样条都分布指定数量的集群。

    • Smooth Rotation
      集群在样条硬转角处的过渡是否更柔和。
      前提：勾选Align Clone。

  • Loop
    偏移时，离开样条的个体是否在另一端循环出现。

  • Rate
    指定一个偏移的速度，让集群持续偏移运动。
    这个与它上边Offset的值无关。

目标指定为粒子发射器对象

对应的是粒子分布。

集群会出现在粒子的位置。

粒子分布自身没有造型基础、集群数量、造型变换（个体关系） 、造型变换（变体） 的参数，可以等效替代为调节目标粒子发射器对象的自身属性。

目标指定为矩阵对象

对应的是矩阵分布。

集群会出现在矩阵点的位置。

矩阵分布自身没有造型基础、集群数量、造型变换（个体关系） 、造型变换（变体） 的参数，可以等效替代为调节目标矩阵对象的自身属性。
