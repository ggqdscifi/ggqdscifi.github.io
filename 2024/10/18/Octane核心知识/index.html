首要设置（在OC的全局设置里）

OC默认的一些设置是不利于制作和渲染的，需要提前修改一下。

Kernal选项卡

  • 第一个下拉菜单
    渲染解算的模式。

    • Info channels
      信息通道。一般用来预览通道。
    • Direct lighting
      直接光照。渲染相对较快，但不够写实。
    • Path tracing、Photon tracing、PMC
      三种无偏渲染模式。三者差别不大，一般用Path tracing路径追踪。

  • Max. samples
    最大采样。值越大，品质越高，速度越慢。预览的话设为800就够，最终渲染的话可以设为2000-20000。

  • Diffuse depth、Specular depth
    漫射、折射的作用程度。值越高，效果越真实，场景越亮。

    • Diffuse depth预览的话设为8就够，最终渲染可以设为16。
    • Specular depth设为8就行，场景里有玻璃等物件的话可以增加。

  • Gl clamp
    对全局光照程度的钳制。降低到1000可以防止出现伪阴影。

  • Alpha channel
    保留透明通道。

  • Static noise
    让噪点不会过分闪动。建议勾选。

  • Adaptive sampling
    根据场景自动优化采样。建议勾选。

Settings 选项卡下的Env. 子选项卡

  • Default environment color
    默认环境光的颜色。一般是改成纯黑，也就是完全没有光了，然后手动打光。

注意黏土模式

  • Clay modes
    黏土模式。忽略反射等材质属性来查看，就像看白模一样，这时候调材质是没反应的。

OC灯光

日光 Daylight Tag

  • Power、Sun intensity
    Power是调整日光的整体强度；Sun intensity只调整迎光面的光强度，也就是主光的强度，暗部不会受影响。

    一般是先调Power定一个整体光照的调，然后调Sun intensity控制主光的强度。
  • North Offset
    沿地平面旋转日光的角度。与直接旋转日光对象的区别是后者会影响光的强度。
  • Sun size
    太阳大小。影响影子的虚实。
  • Sky color
    天空颜色。会影响场景的偏色（因为Octane日光会自带天空）。
  • Mix sky texture
    如果场景里还有OC的HDR环境，勾选可以同时显示日光和HDR光照。

HDR Environment Tag

通过一张HDR贴图形成光照。

区域光 Octane LightTag

  • Texture
    光源纹理。通过Color、Gradient等着色器来改变光源颜色。
  • Distribution
    光源分布。使用贴图（选择c4doctane-ImageTexture）来改变光源的轮廓，还可以导入IES文件来形成射灯效果。
    注意如果是改变区域光对象本身的形状和大小，要到灯光对象的Details选项卡里。
  • Surface brightness
    按面积发光。不勾选的话，光源缩放就不会影响光的强度。
  • Double sided
    双面发光。
  • Visible on diffuse、Visible on specular、Cast shadows
    是否会造成漫射、高光、投影。
  • Opacity
    区域光自身对象的透明度。如果挡画面的话，就调成0。

OC材质

材质的显示顺序：优先显示后添加的材质；优先显示子级对象自身的材质。

材质的选择：透明就用Specular；不透明的话，有反射就用Glossy，没有反射就用Diffuse。

漫射 Octane Diffuse Material

  • Diffuse通道
    材质的底色。

    • Texture
      可以添加纹理贴图。（三个常见的C4D自带基础着色：Color、Gradient、Noise）
      添加贴图要到c4doctane-ImageTexture，并在其中调节贴图的投射方式和变换。

  • Bump通道、Normal通道、Displacement通道
    都是放贴图的，用来添加凹凸质感。
    Bump：用贴图的方式添加凹凸。是视觉上的伪凹凸，但速度快。
    Normal：在Bump的基础上还能带有伪凹凸的角度。
    Displacement：用贴图的方式添加凹凸。是真实的凹凸，但会慢（因为会增加细分面）。

    • Displacement通道中添加着色器后，着色器的Level of detail一般选2048x2048，如果太小效果会很粗糙，太大会卡。

  • Opacity通道
    可以通过贴图隐藏模型的某部分。

  • Emission通道
    自发光。

    • Blackbody emission着色器
      一般做纯色发光体。
    • Texture emission着色器
      可以加贴图，比如做电子显示屏这种。

  • Transmission通道、Medium通道
    用来做SSS半透效果。

光泽 Octane Glossy Material

等于在漫射基础上多了反射的属性。

  • Roughness通道
    粗糙度。控制磨砂质感的程度。
  • Index通道
    折射率。控制整体反射的强度，1<值≤8时，值越大反射越强，越像金属；但值=1的时候是金属质感最强的。

镜面 Octane Specular Material

  • Index通道
    折射率。一般直接输入对应物体的值。（常见物体折射率）

  • Transmission通道、Medium通道
    用来做彩色玻璃效果。
    Transmission：只是单纯对玻璃着色，但颜色不受玻璃厚度的影响，不够真实。
    Medium：更真实的玻璃色彩。

    • Medium通道

      • Absorption着色器

        • Density
          玻璃颜色的浓淡。这个参数数值不是特别敏感，要大幅度改。
        • Absorption
          继续添加着色器来改变玻璃颜色。

      • Scattering着色器
        做半透材质。

节点编辑器

比挨个改参数省事的方式（比如同一种投射方式用一个节点连上就行了）。

基础材质节点

纹理节点

  • Image texture
    贴图。
  • Rgb spectrum
    纯色；也可作为值（有明暗的都可以作为值）。

纹理调整节点

对应的就是在贴图中调节贴图的投射方式和变换。

自带纹理节点

  • Checker
    棋盘格；可作为值。
  • Octane noise
    OC噪波。

调色节点

  • Color correction
    调整色相、饱和度、亮度。
    要连在贴图节点和材质节点中间。
  • Octane gradient
    生成渐变纹理；对贴图调色（用色带上的颜色映射贴图里的明暗区域）。
  • Mix
    混合多个贴图。
